# Houselytics

A machine learning data application built with Python and Streamlit to analyse housing data and predict house prices.

## Project Overview

This project uses historical housing data from Ames, Iowa to analyse key factors affecting house prices and to build a predictive model that estimates property sale prices.

## Dataset

- Housing Prices Dataset (Ames, Iowa)
- Source: Code Institute (Kaggle)

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Streamlit

## Project Status

Project setup and planning phase.
